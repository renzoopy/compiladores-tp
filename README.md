# Diseño de Compiladores

Repositorio https://github.com/renzoopy/compiladores-tp

## Requerimientos

-   Python >= 3.8

## Ejecución

1. Clonar el repositorio:

    ```bash
    git clone git@github.com:renzoopy/compiladores-tp.git
    ```

2. Ejecutar el script:

    ```bash
    python .\main.py
    ```
